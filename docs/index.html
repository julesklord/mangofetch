Overview

Un engine headless, con TUI encima

En el centro vive mangofetch-core: un engine ligero y headless sobre Tokio y Reqwest, con una API basada en traits de Rust para hablar con YouTube, torrents (magnet), SoundCloud, Instagram y el resto de las 1000+ plataformas vía yt-dlp y ffmpeg.

Para todos los demás, mangofetch trae mangofetch-cli: una TUI construida con clap y ratatui, con soporte de mouse, configuración cómoda y 11 paletas de fruta tropical para no ver siempre la misma terminal gris.

Using as a Rust library

O sáltate la TUI y úsalo como SDK

Si estás armando un bot de Discord, un servidor web o tu propia app, mangofetch-core se integra directo en tu código Rust. 

[dependencies]
mangofetch-core = { git = "https://github.com/julesklord/mangofetch-cli" }

Command reference

Comandos principales

Comando Alias corto Qué hace
mangofetch download <url> mango d Descarga un solo link.
mangofetch download-multiple <file> mango dm Descarga varios links desde un archivo.
mangofetch info <url> mango i Ve la info del link sin descargar.
mangofetch list mango ls Muestra lo que hay en la cola.
mangofetch clean mango c Limpia historial y caché.
mangofetch config mango cfg Ajusta límites y rutas.
mangofetch check mango ch Verifica que tus binarios externos funcionen.
mangofetch update mango up Actualiza yt-dlp/ffmpeg a su última versión.
mangofetch logs mango log Revisa logs de la app.
mangofetch about mango a Versión y licencia.

Alias cortos marcados como upcoming. Ver el ROADMAP oficial para detalles.

Core components

Arquitectura del workspace

  • mangofetch-core

    El corazón del sistema — cola de descargas, registry de plataformas y manejo de yt-dlp/ffmpeg.

  • mangofetch-cli

    Frontend con clap y ratatui — rápido, se ve bien, muestra todo en tiempo real.

  • mangofetch-plugin-sdk

    Toolkit para añadir features a mangofetch mientras corre.

CLI/TUI installation

Instalación

Vía Cargo (recomendado)

cargo install mangofetch-cli

Desde el código fuente

git clone https://github.com/julesklord/mangofetch-cli.git
cd mangofetch-cli
cargo build --release
